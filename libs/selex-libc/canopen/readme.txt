Quick guide:
Step 1: Create app_canopen in app layer of your project
Step 2: Copy all folders,files in example_blank and paste to app_canopen folder
Step 3: 
If you are selex guy and love simplicity,
	only edit 3 file:
	app_co_od_simply_config
	app_co_od.c
	app_co_init.c

Else if you want more customization, 
	you can change all the files in app layer carefully after reading the documentation
	(Refer: https://docs.google.com/document/d/1foOrjuBwdzhSF4Z2aK-nWCut4JuYZvoZJQ0KPzVXrFk/edit?usp=sharing)
